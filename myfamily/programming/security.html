Copilotとの対話記録 - インターネット安全対策

🔍 背景

Masatsuyo さんは、中学1年生になる孫にプログラミングを教えようと考えていた。しかし、その前にインターネットの安全対策を講じることが重要だと気づき、Microsoft Copilot に相談することにした。

💬 Copilot との対話

Masatsuyo: Google Meet などを使って孫にプログラミングを教えようと思っていますが、その前にインターネットの安全対策を講じようと思います。
Copilot: 素晴らしいですね！プログラミングを学ぶ前に安全なインターネット利用を理解することは、とても大事です。例えば、個人情報の保護や安全なコミュニケーション、フィルタリングの設定などがありますよ。

✅ インターネット安全対策のポイント

  • 個人情報の保護： 氏名や住所などの情報をインターネット上で共有しない。
  • 安全なコミュニケーション： 知らない人と慎重にやり取りし、不審なリンクは開かない。
  • フィルタリングの活用： Chrome と Edge のセーフサーチ設定を適用。
  • 詐欺対策： フィッシング詐欺や不正アクセスに注意する。
Masatsuyo: ありがとうございます。具体的には Chrome と Edge でセーフサーチを設定し、Windows の Family Safety を活用するつもりです。
Copilot: それはとても良い方法ですね！それぞれの設定方法を詳しく説明します。

🔧 Chrome と Edge のセーフサーチ設定

Google Chrome の設定

  1. Googleの検索設定ページへアクセス。
  2. 「セーフサーチ フィルタ」をオンにする。
  3. 「保存」をクリックして適用。

Microsoft Edge の設定

  1. Bingの検索設定ページへアクセス。
  2. セーフサーチのレベルを「厳密」に設定。
  3. 「保存」をクリックして適用。

👨‍👩‍👧‍👦 Microsoft Family Safety の活用

Masatsuyo さんの家族（じじ・息子・嫁・孫）が関係するため、役割分担を考慮して設定を進めることが重要。

家族の役割

役割 主な責務
じじ（Masatsuyo） 初期設定のサポート、アドバイス
息子 & 嫁 管理者としてルールの決定と監視
孫 決められたルールに従い、安全にインターネットを利用

設定手順

  1. Microsoft Family Safety のファミリーグループを作成。
  2. 息子さんと嫁さんを「管理者」として追加。
  3. お孫さんのアカウントを「子供」として設定し、利用時間やフィルタリングを適用。
